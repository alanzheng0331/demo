不是Spring Boot项目，配置数据库连接。

创建实体类User，对应数据库user表。

创建UserMapper接口，用于数据库操作。

创建UserService，实现注册和登录逻辑。

创建UserController，包含注册和求职者登录接口。

创建AdminController，包含管理员登录接口。


HTTP请求 → Controller层 → Service层 → Mapper层 → 数据库
    ↓          ↓           ↓          ↓        ↓
前端/客户端 → UserController → UserService → UserMapper → user表
